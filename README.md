# base64-font-extractor

A simple CSS Base 64 font extractor.

You NEED to have Font Forge installed and available on `PATH` system ENV variables, to be able to use this tool.

# Usage

`node ./src/index.js download https://www.example/<path-to-a-css-file>.css`
